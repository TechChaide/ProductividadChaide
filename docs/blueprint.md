# **App Name**: Order Lookup

## Core Features:

- Operator Code Input: Input field for operator code to authenticate and display employee picture
- Orders List: Display list of order that comes from a public api.
- Single Order Selection: Enable single row selection for the orders table.
- Display Selected Order: Displays selected order from the orders table.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to create a calm and reliable feel.
- Background color: Very light blue (#F0F8FF), almost white, provides a clean backdrop.
- Accent color: Light purple (#B19CD9) for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a clean and neutral appearance for both headers and body text.
- Use simple, outline-style icons to match the minimalist aesthetic.
- Emphasize whitespace and clear delineation between sections.
- Use subtle transitions and animations when loading and selecting orders.
